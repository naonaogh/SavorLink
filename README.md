# SavorLink 
SavorLink — веб-платформа для упрощения взаимодействия ресторанов и кафе с поставщиками сырья в России и Крыму.

Платформа позволяет:
- Находить и сравнивать поставщиков по цене, условиям и рейтингу.
- Просматривать карточки поставщиков с товарами, ценами и отзывами.
- Создавать и оформлять заказы, использовать шаблоны для повторяющихся закупок.
- Управлять своим профилем и товарами, вести историю заказов.
- Общаться с поставщиками и покупателями через встроенный чат.
- Отслеживать статус заказов и приёмку товаров.
- Генерировать документы (счета, контракты, акты) автоматически.
- Получать аналитику по заказам, спросу и поставщикам.
- Использовать избранное и уведомления о новых событиях.

SavorLink делает процесс закупок простым, быстрым и прозрачным для всех участников.

Технологии: 
Python 3.12
FastAPI (backend)
SQLAlchemy + PostgreSQL (БД)
Alembic (миграции БД)
MinIO (хранение документов и файлов)
LaTeX (формирование PDF-отчётов)
Poetry (управление зависимостями)
